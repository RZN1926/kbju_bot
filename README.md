# 🍎 KBJU Tracking Bot

Телеграм-бот для мониторинга питания (КБЖУ), синхронизированный с веб-интерфейсом Firestore.

---

## 🚀 Функционал
- 📊 **Трекинг КБЖУ:** Добавление приемов пищи через FSM.
- 🥗 **Каталог:** Управление личной базой продуктов.
- 🔗 **Синхронизация:** Единая база данных с веб-версией.
- 📅 **История:** Просмотр статистики за выбранные дни.

---

## 🛠 Стек
- **Language:** Python 3.12
- **Framework:** aiogram 3.x
- **Database:** Google Firebase (Firestore)
